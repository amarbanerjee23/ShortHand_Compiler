# ShortHand Semantics

ShortHand uses block scopes with no redeclaration in the same scope. Conditions are true when nonzero for legacy integer programs; the specified boolean semantics prefer `bool` once semantic analysis is complete. Arrays are fixed-size and must be indexed by integers; static bounds should be rejected and dynamic bounds checked. `break` and `continue` are legal only in loops. `return` exits the current function frame.

`greenai(name, inferences, watts, seconds)` records an evidence event; negative watts or seconds are invalid. `ai_infer(model_path, shape_csv, input_csv)` calls the optional runtime. Without ONNX Runtime, it must emit a clear fallback error instead of pretending inference succeeded.
