# Deprecated Python Green AI tooling

The official ShortHand Green AI workflow is now the compiled C++ binary
`Compiler_new_ws/Short_Hand/build/green_ai_tool` built with:

```bash
make -C Compiler_new_ws/Short_Hand/src green_ai_tool
```

This directory is retained only for historical migration reference. It is not
part of the supported toolchain, tests, examples, or documentation path.
